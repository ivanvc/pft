h1. pft!

A command line twitter client. It's intended to be as powerful as donkey.

h2. Requirements

* ruby
* rubyems
* Included in the gem:
** twitter
** choice (0.1.3.1)
** highline
** rspec

h2. Installation

Just hit

@gem install ivanvc-pft --source http://gems.github.com/@

Then you're ready to go. You should have the pft binary.

h2. Help

Use @pft -@ or @pft --help@
